A fork (or sorts) of Omnik-Data-Logger (https://github.com/clmcavaney/Omnik-Data-Logger.git) to get data from a local Enphase Envoy device and log it to one or more outputs.

Outputs can be:
- Console
- CSV
- MQTT
- MySQL
- PVOutput
